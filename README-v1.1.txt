EVE Chat v1.1 — 通用角色贴合引擎＋萧逸专属角色包
====================================================

本版本基于 EVEChat v1.0（动态连续回复＋网页图标）继续开发。
原有天气、主动聊天、记忆、时间线、撤回、表情包、动态、通知、网页图标功能均保留。

主要功能
--------
1. 通用角色贴合引擎
   - 按角色ID绑定档案，避免污染其他角色
   - 也可按角色名称自动匹配
   - 区分普通聊天、主动消息、动态正文、动态评论和剧情模式
   - 每轮只挑选相关示例，不把所有资料都塞进Prompt
   - 本地OOC检测
   - 角色档案导入／导出
   - 可创建简单的通用角色档案

2. 萧逸专属深度角色包
   - 深层性格、行为逻辑与防御方式
   - 萧逸与夕月的固定关系层
   - 日常聊天、动态与赛车场景独立语言模板
   - 陈述句句尾句号／感叹号程序修正
   - 心理咨询师化、病娇化、霸总化、过度顺从等OOC检测
   - R1五冠王职业能力层

3. R1实时赛事
   - 使用Jolpica F1 API读取下一场F1赛历，作为R1赛历原型
   - 可使用OpenF1补充练习、排位、冲刺与正赛Session信息
   - 现实车手与车队名称默认不会进入萧逸世界
   - 现实比赛结果不会自动等同于萧逸成绩
   - 真实实时数据可用性取决于数据源；OpenF1实时数据可能需要订阅

4. 生成参数
   - 萧逸档案启用时，默认建议 temperature 1.18、topP 0.93、topK 50
   - 只在该角色档案生效，不影响其他角色

设置位置
--------
打开任一角色聊天室：

聊天设置 → EVE扩展功能 → 角色贴合增强

可设置：
- 启用／关闭角色贴合
- 按角色ID绑定档案
- 自动匹配角色名称
- 普通／严格／极严格
- 句尾标点程序修正
- 本地OOC检测
- 是否使用角色推荐生成参数
- 每轮参考示例数量
- R1实时赛事设置
- 创建、导入与导出通用角色档案

萧逸绑定
--------
如果角色名称为“萧逸”或“Osborn”，系统会自动识别。
更稳妥的做法：

1. 打开萧逸聊天室
2. 打开“角色贴合增强”
3. 在“绑定角色档案”选择“萧逸・夕月专属版”
4. 点击“保存绑定”

这样即使以后修改角色显示名称，仍会按角色ID使用萧逸专属包。

资料版本说明
------------
- 当前锁定：萧逸是R1五冠王
- 图片资料中出现“五连冠”，文字资料明确为“五冠王”
- 本版本只锁定获得五次冠军，不强制认定连续五冠
- 仓库初遇年龄在资料中出现11岁／13岁两个版本，本版本使用“少年时期”避免擅自选边

限制
----
- v1.1先做本地OOC检测与语言后处理，不默认额外调用一次API重写，避免增加费用与延迟
- 通用角色档案目前是手动整理版；“上传资料后由AI自动分析成档案”可在后续版本增加
- R1实时赛事需要联网；网络失败时不会影响原本聊天
- OpenF1历史资料免费，真正实时资料可能需要付费订阅
