EVE Chat Relationship Module v0.1.0
===================================

檔案：js/relationship.js
Adapter：v0.5.0

資料維度
- affection：好感
- trust：信任
- familiarity：熟悉
- attachment：依賴

關係階段
stranger → acquaintance → friendly → close → intimate → deep-bond

自動事件
- eve:user-message-sent：暫存使用者訊息
- eve:ai-message-received：完成一次對話後計算關係變化
- eve:proactive-dispatch-complete：記錄角色主動聯絡
- eve:timeline-added：重要共同事件小幅增加關係
- eve:adapter-ready：註冊 Prompt Context Provider

公開 API
EVERelationship.getState()
EVERelationship.update({ affection, trust, familiarity, attachment })
EVERelationship.set({...})
EVERelationship.getPromptContext()
EVERelationship.getStats()
EVERelationship.exportData()
EVERelationship.importData(data)
EVERelationship.clear()
